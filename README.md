#About#

An LLVM based verification framework.

#Compilation#

* ``` mkdir build ```
* ``` cd build ```
* ``` cmake -DCMAKE_BUILD_TYPE=Release ../ ``` or ```cmake -DCMAKE_BUILD_TYPE=Release ../ -GNinja```
* ```make``` or ```ninja```
